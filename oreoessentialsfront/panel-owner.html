Plugin Setup

How to connect OreoEssentials to the web panel.

⚙️ 1. Get Your API Key

Register a server in Add Server. After creation, you'll see a one-time API key to copy.

📝 2. Add to plugin config.yml

web-panel:
  enabled: true
  url: http://localhost:8080   # your panel URL
  api-key: oreo_abc123_xyz789…
  sync-interval-ticks: 6000    # every 5 minutes (default)
  sync-on-login: true
  sync-on-logout: true

🔄 3. How Data Sync Works

The plugin sends player data to the panel API on:

  • Login — pushes a snapshot when a player joins
  • Logout — final sync on quit
  • Interval — every N ticks (configurable)
  • Balance change — real-time economy updates

Players must link their web account (via Panel Login) and enter your server slug to see their data.

🌐 4. Direct MongoDB Mode

If you set mode to DIRECT_MONGO, no plugin changes are needed. The panel reads your MongoDB directly. Requires your MongoDB to be network-accessible from the panel server.

🔗 5. Player Verification

After a player links their web account to your server slug, their link is verified automatically on the first sync for that UUID. No extra steps needed on your side.
