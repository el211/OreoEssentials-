🎄 Installation & Setup

Requirements

  • Minecraft 1.20+
  • Paper / Purpur
  • Vault (economy)
  • LuckPerms
  • LibsDisguises (mandatory)
  • ItemsAdder (optional, custom icons)
  • MythicMobs support (optional)
  • OreoEssentials support (optional)

Installation

  1. Drop UltimateChristmas.jar into /plugins/.
  2. Restart the server.
  3. Edit configuration files:
    • config.yml
    • daily_gifts.yml
    • santa.yml
    • shardshop.yml
  4. Run /ultimatechristmas reload to apply changes.
