⚙️ General Commands & Permissions

UltimateChristmas includes modules for Advent, Shards, Santa, Snowball, optional Grinch boss, and Snow controls.

🏠 Main Command

CommandDescriptionPermissionDefault
/ultimatechristmas reloadReload all configsultimatechristmas.admin.reloadOP
/ultimatechristmas versionShow plugin versionultimatechristmas.infotrue

🎁 Advent Calendar

CommandDescriptionPermissionDefault
/adventOpen Advent GUIultimatechristmas.adventtrue
/advent claimClaim today’s giftultimatechristmas.adventtrue
/advent remainingTime until resetultimatechristmas.adventtrue
/advent streakShow current streakultimatechristmas.adventtrue
/advent open <player>Open for another playerultimatechristmas.advent.adminOP
/advent claim <player>Force-claim for playerultimatechristmas.advent.adminOP

Permissions

  • ultimatechristmas.advent — access Advent (default: true)
  • ultimatechristmas.advent.admin — manage others (default: OP)

CommandDescriptionPermissionDefault

🌨️ Global Snow Controls (overlay / region-aware)

These mirror your “snow overlay” settings (roof respect, blocked WG regions, radius, density).

CommandDescriptionPermissionDefault
/snow onEnable global snow overlayultimatechristmas.snow.adminOP
/snow offDisable global snow overlayultimatechristmas.snow.adminOP
/snow radius <blocks>Set effect radiusultimatechristmas.snow.adminOP
/snow density <0..100>Set visual flake densityultimatechristmas.snow.adminOP
/snow reloadReload snow.yml / configultimatechristmas.snow.adminOP
/snow me <on|off>Player toggle (if allowed)ultimatechristmas.snow.playertrue

🧰 Admin & Utility

  • ultimatechristmas.* — all features (OP)
  • ultimatechristmas.admin.reload — reload configs (OP)
  • ultimatechristmas.admin.debug — debug logs (OP)
  • ultimatechristmas.info — version/info (true)
