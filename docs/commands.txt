# Commands & Permissions — Complete Reference

Every command in OreoEssentials, sourced directly from `plugin.yml`.

**Permission defaults:**
- `true` = all players have this by default
- `op` = only server operators have this by default
- `false` = nobody has this by default (you must grant it manually in LuckPerms)

**Corrections vs previous draft:**
- `/tpaccept` permission is `oreo.tpaccept` (NOT `oreo.tpa`)
- `/tpdeny` permission is `oreo.tpdeny` (NOT `oreo.tpa`)
- `/tphere` permission is `oreo.tphere`, default `op`
- `/tp` permission is `oreo.tp`, default `op`
- Many commands were missing entirely — all are now included
